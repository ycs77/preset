### Laravel Presets

| Preset                                                              | Install                                      |
| ------------------------------------------------------------------- | -------------------------------------------- |
| [Initialize](https://github.com/ycs77/preset-laravel)               | `npx apply ycs77/preset-laravel`             |

| Inertia                                                             | `npx apply ycs77/preset-laravel-inertia`     |
